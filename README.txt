Το παρόν project είναι ένα CI/CD pipeline που χρησιμοποιεί το Jenkins για την αυτοματοποίηση της δημιουργίας υποδομών στο AWS μέσω Terraform. Το pipeline αποτελείται από τέσσερα στάδια: Checkout, που κατεβάζει τον Terraform κώδικα από το GitHub, InitPlan, που αρχικοποιεί το Terraform και δημιουργεί ένα σχέδιο υποδομής (plan), Approval, που ζητά επιβεβαίωση αν το autoApprove είναι false, και Apply, που εφαρμόζει το σχέδιο για να δημιουργήσει την υποδομή στο AWS. Η υποδομή περιλαμβάνει μία VPC, ένα Subnet, ένα Security Group για SSH/HTTP και ένα EC2 instance. Τα AWS credentials διαχειρίζονται με ασφάλεια μέσω του Jenkins.


This project is a CI/CD pipeline that uses Jenkins to automate the creation of infrastructure on AWS through Terraform. The pipeline consists of four stages: Checkout, which downloads the Terraform code from GitHub; InitPlan, which initializes Terraform and creates an infrastructure plan; Approval, which requests confirmation if autoApprove is set to false; and Apply, which applies the plan to create the infrastructure on AWS. The infrastructure includes a VPC, a Subnet, a Security Group for SSH/HTTP, and an EC2 instance. AWS credentials are securely managed through Jenkins.
